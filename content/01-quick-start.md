# Quick start
## Global installation
```bash
> npm i -g create-react-app
> create-react-app my-app
> cd my-app
> npm start
```
## Update
```bash
> update -g create-react-app
```
## Local installation
```bash
> npx create-react-app my-app
> cd my-app
> npm start
```
